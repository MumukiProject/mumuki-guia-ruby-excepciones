Recordemos a nuestra vieja amiga `pepita` y su método `volar_en_circulos!`. Ahora _evolucionó_ y es un objeto de la clase `Golondrina`:

```ruby
class Golondrina
  def initialize
    @energia = 50
  end
  
  def volar_en_circulos!
    @energia -= 20
  end
end

pepita = Golondrina.new

```
